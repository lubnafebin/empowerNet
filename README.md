## EmpowerNet — Frontend
### EmpowerNet is a digital platform designed to streamline the management of Self-Help Groups (SHGs) similar to Kudumbashree in Kerala. The frontend is built to provide a clean, responsive, and efficient user experience for CDS, ADS, NHG, and Member-level users.
## Features
<ul>
  <li>Role-based Authentication (CDS, ADS, NHG, Member)</li>
  <li>Dashboard Interface for quick insights</li>
  <li>Meeting Management — schedules, attendance, and reports</li>
  <li>Financial Tracking — savings, loans, and expenses</li>
  <li>Member Profiles — personal + financial details</li>
  <li>Reports Module — downloadable and auto-generated</li>
  <li>Fully Responsive UI built with React & Tailwind</li>
</ul>
